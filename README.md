### Run Locally

To run **ZenTask** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/teddylumidi/ZenTask.git
```

Windows:

```bash
git clone https://github.com/teddylumidi/ZenTask.git
```
